Data Science Projects

Exploring Clarity & Reach in Scientific Communication

Does the use of scientific jargon affect the number of citations that a research paper will receive? Do certain topics receive greater engagement on Medium's Towards Data Science blog? For my capstone project, I explored the relationship between clarity, content, and reach in scientific communication, using topic modeling with Latent Dirichlet Allocation (LDA) and other Natural Language Processing (NLP) techniques.

Kiva.org MaxApprover App

I worked with a team of data scientists on a pro bono project to develop, execute, and implement an interactive borrower screening tool designed to improve efficiency for a microfinance institution using machine learning models and NLP techniques.

Reddit NLP Classification App

Using over 100K posts from the r/books and r/writing subreddits (collected with the pushshift.io api), I developed, trained, tested, and compared several classification models using the machine learning and NLP tools in scikit-learn and NLTK. The best-performing model correctly identified the subreddit of unseen posts with 89.7% accuracy, using a Logistic Regression estimator and a Term Frequency - Inverse Document Frequency transformer.
